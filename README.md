# Element-ex
element ui & extend

# Installation
```shell
npm install element-ex
```

#usage
```
import ElementEX from 'element-ex'
import 'element-ex/dist/styles/element-ex.css'
Vue.use(ElementEx)
```

#Build
```shell
npm run dist
```
# Example
```javascript
npm run dev
```
